## Build & Customize

1. Install dependencies with `npm install` or `pnpm install`.
2. Run `npm run dev` to launch the local development server.
3. Commit and deploy through your preferred workflow (e.g. Vercel).

## How It Works

1. Develop new features locally in branches.
2. Open pull requests for review and QA.
3. Merge approved changes into the main branch.
4. Trigger deployments automatically via your hosting provider.
